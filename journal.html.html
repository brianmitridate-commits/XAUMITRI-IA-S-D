🔗 Connexion Google Sheets
Synchronise ton journal automatiquement avec Google Sheets. Accessible partout, sur tous tes appareils.
1
Ouvre Google Sheets → cree une feuille "SD Journal Trading"
Puis Extensions → Apps Script
2
Colle le code du fichier apps_script.gs et clique Enregistrer
3
Clique Deployer → Nouveau deploiement → Application Web
Acces : "Tout le monde" → Deployer → Copie l'URL
URL apparaitra ici apres sauvegarde
4
Colle l'URL ci-dessus et clique Sauvegarder.
Ton journal se synchronisera automatiquement a chaque trade.
💾 Import depuis l'outil principal
Exporte le JSON depuis l'outil Avocat du Diable et importe-le ici pour tout retrouver.
⚠ Zone dangereuse
Ces actions sont irreversibles.
